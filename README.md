# 🛒 Ecommerce Project

A simple PHP-based Ecommerce web application with Admin Panel, Product Management, and User Authentication.

---

## 📁 Project Structure

Ecommerce/
│
├── admin/              # Admin panel (dashboard, login, logout, products)
├── css/               # Stylesheets
├── images/            # Product and UI images
├── includes/          # Reusable PHP components (db, header, footer)
├── pages/             # Frontend pages (shop, cart, etc.)
│
├── index.php          # Main landing page
├── databaseSetup.md   # Database setup guide
├── loginPageSetup.md
├── registrationPageSetup.md
├── adminLoginPage.md
├── adminDashboardPage.md
├── adminAddProductPage.md
├── adminManageProductsPage.md
├── adminLogoutPage.md
├── cartPageSetup.md
└── README.md

---

## ⚙️ Features

- User Registration & Login
- Admin Login System
- Admin Dashboard
- Add / Manage Products
- Shopping Cart System
- Logout System with session handling
- Structured folder system

---

## 🛠️ Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript

---

## 🗄️ Database Setup

1. Create database in phpMyAdmin  
2. Import SQL file  
3. Update DB config in `includes/db.php`

---

## 🚀 How to Run

git clone https://github.com/your-username/Ecommerce.git

Move project to:
htdocs/Ecommerce

Start XAMPP (Apache + MySQL)

Open in browser:
http://localhost/Ecommerce

---

## 🔐 Admin Access

Login:
http://localhost/Ecommerce/admin/login.php

Make sure admin user exists in database.
